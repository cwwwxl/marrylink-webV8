# 首页工作台说明文档

## 功能概述

首页根据用户角色（`userType`）展示不同的内容：

### 1. 主持人工作台 (userType === 'HOST')

主持人登录后将看到专业的工作台界面，包含以下模块：

#### 1.1 顶部问候卡片
- 根据时间显示不同的问候语（早上好、下午好等）
- 显示主持人姓名和当前日期
- 蓝色渐变背景，体现专业感

#### 1.2 统计数据卡片（2x2网格布局）
- **本月订单**：显示本月订单数量和环比趋势
- **待处理问卷**：显示待处理问卷数量（红色高亮提醒）
- **本月收入**：显示本月收入金额和目标完成率
- **满意度评分**：显示平均评分和评价数量

#### 1.3 快捷功能入口
- 档期管理（蓝色）
- 问卷管理（紫色）
- 我的订单（绿色）
- 更多功能（橙色）

#### 1.4 待处理事项列表
- 显示最近3条待处理的问卷或任务
- 每项显示图标、标题、描述、时间和紧急程度标签
- 点击可跳转到详情页

#### 1.5 本月档期日历
- 完整的月历视图
- 支持上一月/下一月切换
- 日期状态标识：
  - 今天：浅蓝色背景
  - 已占用（occupied）：深蓝色背景
  - 已锁定（locked）：黄色背景
  - 有事件：底部显示小圆点

#### 1.6 即将到来的婚礼列表
- 显示最近3场婚礼
- 每项显示日期、新人姓名、场地、时间和状态
- 状态标签：已确认（绿色）、沟通中（蓝色）、待确认（黄色）

### 2. 新人端首页 (userType !== 'HOST')

新人登录后看到的是原有的首页内容：

#### 2.1 顶部轮播图
- 展示婚礼相关的宣传图片

#### 2.2 快捷入口
- 找主持人
- 填写问卷
- 我的预约
- 客服咨询

#### 2.3 推荐主持人
- 横向滚动列表
- 显示主持人头像、姓名、标签、评分和经验

#### 2.4 精彩案例
- 2列网格布局
- 显示婚礼案例图片、标题和日期

## API接口

### 主持人工作台接口

所有接口定义在 [`dashboard.js`](../api/dashboard.js) 中：

1. **获取统计数据**
   - 接口：`GET /host/dashboard/stats`
   - 返回：本月订单、收入、评分、待处理问卷等统计数据

2. **获取待处理问卷**
   - 接口：`GET /host/dashboard/questionnaires`
   - 参数：`{ current, size }`
   - 返回：待处理问卷列表

3. **获取本月档期**
   - 接口：`GET /host/dashboard/schedule`
   - 参数：`{ year, month }`
   - 返回：本月档期数据（日期和状态）

4. **获取即将到来的婚礼**
   - 接口：`GET /host/dashboard/upcoming-weddings`
   - 参数：`{ current, size }`
   - 返回：即将到来的婚礼列表

## 模拟数据

当API接口尚未实现时，代码会自动使用模拟数据：

- **统计数据**：本月订单8个，收入¥32,800，评分4.9
- **待处理问卷**：3条模拟问卷
- **档期数据**：6月9日、10日、16日、25日有安排
- **即将婚礼**：3场婚礼（6月20日、25日，7月1日）

## 样式主题

遵循MarryLink的蓝白主题：

- **主色调**：`#1d4ed8`（深蓝）、`#3b82f6`（亮蓝）
- **背景色**：`#ffffff`（白色）、`#f5f7fa`（浅灰）
- **文字颜色**：`#333333`（深灰）、`#666666`（中灰）、`#999999`（浅灰）
- **状态颜色**：
  - 成功/收入：`#10b981`（绿色）
  - 警告/待处理：`#f59e0b`（橙色）
  - 错误/紧急：`#ef4444`（红色）

## 注意事项

1. **权限控制**：确保后端API有正确的权限验证，只有主持人能访问工作台接口
2. **数据刷新**：每次进入页面（`onShow`）都会重新加载数据
3. **错误处理**：API调用失败时会自动使用模拟数据，不影响用户体验
4. **性能优化**：使用`Promise.all`并行加载多个接口，提升加载速度
5. **响应式设计**：所有尺寸使用rpx单位，适配不同屏幕尺寸
